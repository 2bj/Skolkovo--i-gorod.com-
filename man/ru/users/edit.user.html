Идентификационные данные

Здесь вы можете изменить в соответствующих полях Логин и Пароль. Поля являются обязательными для заполнения.

Если вы уже создавали группы Пользователей раньше, то можете внести Пользователя в одну или несколько этих групп. Для этого поставьте галочку напротив соответствующего названия группы.

Если вы не хотите, чтобы данный пользователь мог пользоваться системой, снимите галочку Активрован. В таком случае Пользователь будет будет заблокирован. Вы можете активировать его учетную запись в любой момент.

Персональные данные

По необходимости вы можете изменить или дополнить поля с личными данными пользователя: Фамилия, Имя, Отчество и т.п.

Права доступа

Вы можете изменить набор прав, поставив дополнительные галочки. Следует иметь в виду, что права пользователя не могут быть меньше, чем права группы, в которую он входит.

Подробнее о настройке прав

Вы можете самостоятельно расширять и редактировать набор полей, необходимых для заполнения при создании/редактировании Пользователя. Для этих целей существует модуль Шаблоны данных.
