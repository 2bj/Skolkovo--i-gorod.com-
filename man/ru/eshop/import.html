Импорт данных в каталог UMI.CMS осуществляется с использованием следующих форматов:

  • CSV (comma separated values)
  • CML 1.0 (Commerce Markup Language 1 version)
  • CML 2.0 (Commerce Markup Language 2 version)
  • UMIDump 1.0

Вы можете самостоятельно создавать такие файлы или выгружать их из 1С, MS Excel, UMI.CMS при помощи специальных инструментов.

Формат загружаемых данных определяется в автоматическом режиме по содержимому файла. Если в файле отсутствует XML заголовок, то он интерпретируется по правилам CSV формата. Для XML файлов определение формата осуществляется с использованием XSD схемы загружаемого документа. Авто-определение формата файла работает в 99% случаев, при необходимости можно указать формат данных принудительно выбрав его из списка форматов файлов.

CML1, CML2 данные загружаются в корень структуры сайта в разделы «1C CML1 imported», «1C CML2 imported» соответственно.

Для загрузки каталога с ценами из "1С: Предприятия" необходимо сначала загрузить каталог, а затем — прайс-лист.

Рекомендуется использовать для всех поддерживаемых форматов кодировку UTF-8. Исключением являются CSV файлы, которые можно загружать как в UTF-8, так и в кодировке windows-1251.

Подробнее о форматах загружаемых файлов смотрите в нашей документации.
