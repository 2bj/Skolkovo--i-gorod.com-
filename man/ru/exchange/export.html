В этой вкладке показан список имеющихся схем для экспорта. Здесь же можно добавить новую схему, или выполнить вручную экспорт по любой из схем, воспользовавшись ссылкой «Выполнить экспорт».

Нажатие на ссылку «Выполнить экспорт» вызовет диалог, где можно выбрать результат экспорта: либо ссылка на результат экспорта, либо скачиваемый файл с результатом.

Ссылка на результат экспорта может быть использована для обмена данными с внешними ресурсами. Для того, чтобы был возможен такой обмен, необходимо запрашивать результаты от имени пользователя с правами на экспорт.

Список схем может также быть экспортирован или импортирован в формате CSV.
