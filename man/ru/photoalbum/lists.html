Фотогалерея — это виртуальный фотоальбом с фотографиями. Вы можете группировать фотографии по различным признакам: друзья, события, концерты, а затем выкладывать их на сайте.

Созданные фотогалереи располагаются древовидным списком в таблице на этой вкладке. Вы можете раскрыть или скрыть содержимое при помощи пиктограмм и .

Каждый фотоальбом, как и любой другой объект сайта, можно редактировать, скрывать с сайта и удалять при помощи соответствующих всплывающих пиктограмм.

Чтобы создать новый альбом, нажмите на ссылку «Добавить альбом». Если в системе доступны разные типы альбомов, можно заранее выбрать тип из списка, появляющегося при наведении курсора на ссылку.

Добавление фотографий

При выборе фотоальбома, в него можно добавить фотографии несколькими способами:

  • при помощи диалога загрузки, который появится под таблицей снизу при выборе альбома
  • при помощи ссылки «Добавить фотографию» сверху, которая появится над таблицей при выборе альбома
  • при помощи диалога загрузки фотографий из архива по ссылке сверху, которая появится над таблицей при выборе альбома

Ссылка «Добавить кнопку в Picasa» предназначена для интеграции модуля c Google Picasa™.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
