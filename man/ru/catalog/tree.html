Эта страница позволяет управлять иерархией объектов каталога. К объектам каталога относятся разделы и подразделы, а также собственно товары.

Здесь вы можете видеть таблицу со списком объектов каталога. Для добавления поля в таблицу нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов.

Нажатие на откроет список дочерних разделов, где конечными элементами дерева являются товары текущего раздела. Чтобы редактировать свойства объекта, воспользуйтесь всплывающей пиктограммой или нажмите на ссылку с названия объекта.

Ссылка «Добавить раздел»

Нажатие на эту ссылку приведет к созданию нового подраздела на том уровне, где вы сейчас находитесь, и открытию окна для редактирования свойств создаваемого раздела.

Ссылка «Добавить объект»

Нажатие на эту ссылку приведет к созданию нового объекта на том уровне, где вы сейчас находитесь, и открытию окна для редактирования свойств создаваемого объекта каталога.

Также для добавления элементов можно воспользоваться всплывающей пиктограммой . Если в системе доступны разные типы элементов, можно заранее выбрать тип из списка, появляющегося при наведении курсора на ссылку «Добавить».

Один товар может быть помещен в несколько категорий для удобства посетителей сайта. Например, детские сиденья для автомобилей можно поместить как в автомобильные товары, так и в товары для детей. Разные люди будут искать по-разному. Для этого создайте виртуальную копию страницы с товаром из дерева сайта. Подробнее об этом читайте в разделе Управление структурой сайта.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
