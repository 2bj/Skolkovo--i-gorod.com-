Модуль предназначен для создания на сайте форумов, в которых пользователи могут обсуждать интересующие их темы. На основе этого модуля можно создать и гостевую книгу.

В списке конференций можно видеть сами конференции в виде разделов, темы в виде подразделов и сами сообщения. Вы можете раскрыть или скрыть раздел при помощи пиктограмм и .

Поля в таблице можно также добавлять самостоятельно, — для этого нужно нажать на пиктограмму и выбрать нужное поле. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из полей.

При помощи всплывающих пиктограмм можно просмотреть, скрыть/открыть, удалить или добавить подраздел или сообщение.

В зависимости от положения на дереве вы можете добавить конференцию, тему, или отдельное сообщение при помощи соответствующих ссылок над деревом. Если в системе доступны разные типы элементов, можно заранее выбрать тип из списка, появляющегося при наведении курсора на ссылку.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
