Наверху расположена ссылка для добавления новой публикации и выпадающий список блогов, в который можно поместить публикацию.

Ниже помещено дерево постов и комментариев к ним того домена системы, в котором мы находимся. Таблица с деревом может быть отсортирована по любому из полей. Для добавления поля нужно кликнуть на пиктограмму в правом углу.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
