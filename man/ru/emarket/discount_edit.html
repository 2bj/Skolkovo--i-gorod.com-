В секции Свойства скидки можно отредактировать название и описание скидки или изменить активность скидки.

В следущей секции можно изменить «процент скидки», или «фиксированный размер скидки» (только для скидок на заказы).

Набор секций ниже зависит от выбора типа скидки и от набора параметров валидации скидки, выбранных при добавлении.

  • На группу пользователей — выберите группу, для которой будет действовать эта скидка.
  • На пользователей — выберите пользователей из списка, для которых будет действовать эта скидка.
  • На сумму заказа — для скидок на заказ укажите диапазон, в пределах которого будет действовать эта скидка.
  • На сумму покупок пользователя — скидка будет установлена только для пользователей, чья общая сумма заказов больше указанной суммы, с учетом всей истории заказов пользователя.
  • На определенные товары — для скидок на товар выберите из дерева сайта раздел каталога или отдельные товары, на которые будет действовать эта скидка.
  • На связанные товары — для скидок на товар выберите товары, наличие которых в корзине необходимо для действия этой скидки.
  • На временной промежуток — установите временной промежуток действия этой скидки.
