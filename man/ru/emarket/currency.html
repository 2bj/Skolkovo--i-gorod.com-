Модуль позволяет использовать несколько валют в магазине. Для изменения списка валют редактируйте содержимое соответствующего справочника, либо воспользуйтесь ссылкой «Добавить валюту».

Заказы в магазине оформляются в валюте, используемой по умолчанию. Однако, если есть возможность выбрать валюту, то в заказе указывается курс валюты на момент оформления заказа (курсы валют обновляются автоматически).

Изначально валютой по умолчанию является рубль. До первого запуска магазина возможно изменить валюту по умолчанию.

Если пользователь совершил заказ в какой-либо валюте, то система запомнит ее, и при следующем посещении магазина этим пользователем будет использоваться валюта, выбранная им в предыдущем заказе.

Список валют может быть отсортирован по любому из столбцов, а также экспортирован или импортирован в формате CSV.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
