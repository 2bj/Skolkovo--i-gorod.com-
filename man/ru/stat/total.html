По умолчанию показываются данные за последние сутки. Но вы можете изменить временной промежуток и задать даты, пользуясь выпадающими списками, и затем нажать кнопку Отфильтровать.

Для просмотра результатов анализа доступны следующие секции:

  • Облако тегов — в этой секции показаны наиболее популярные теги, которые "собрали" посетители за весь подотчетный период. Число, которое отображается в скобках, — общее количество вхождений для этого тега для всех пользователей. Подробнее о тегах читайте в топике Как пользоваться тегами.
  • Секция Посещения — в этой секции показана статистика посещений выбранных доменов за заданный период. Тут можно посмотреть данные о хитах (показах страниц), хостах (уникальных ip-адресах) и сессиях. Также здесь показывается среднее время пребывания на сайте и среднее количество просмотренных страниц (глубина просмотра) за один сеанс.
  • Секция Источники — здесь собраны максимальные значения по ряду параметров, таких как источник перехода, поисковое слово, поисковая система.
