На этой странице вы можете изменить Название блога.

Также можно отредактировать Список друзей. В этом списке вы можете выбрать тех пользователей, которые будут являться вашими друзьями. В постах можно будет выставлять статус доступа «для друзей» — такие посты будут видны только друзьям. Внимание — если добавите Гостя в друзья, то пост «только для друзей» будут видеть все.

Остальные поля являются стандартными для страниц UMI.CMS.
