В этой вкладке можно видеть список всех баннеров, которые есть на сайте.

В этом списке можно посмотреть все необходимые поля каждого баннера. Для добавления поля нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов, а также экспортирован или импортирован в формате CSV.

Чтобы изменить свойства баннера, нужно нажать на ссылку с его названия или на всплывающую пиктограмму .

Чтобы создать новый баннер, наверху списка есть ссылка «Добавить баннер». Тип баннера можно выбрать заранее из списка, появляющегося при наведении курсора на ссылку.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
