Идентификационные данные

Задайте в соответствующих полях Логин и Пароль. Новый пользователь будет использовать эту учетную запись для доступа в систему. Поля являются обязательными для заполнения.

Если вы уже создавали группы Пользователей раньше, то можете сразу внести нового пользователя в одну или несколько этих групп. Для этого поставьте галочку напротив соответствующего названия группы.

Кнопка-флажок Активирован по умолчанию включена. Если вы не хотите, чтобы данный Пользователь мог немедленно пользоваться системой после его создания, снимите галочку. В таком случае пользователь будет создан, но его учетная запись будет заблокирована. Далее вы сможете активировать его учетную запись в любой момент.

Персональные данные

В этой секции необходимо заполнить поле Имя.

Кроме того, вы можете заполнить поля Фамилия и Отчество.

Права доступа

Отметьте галочками необходимые для данного пользователя права. В случае, если вы включили его в уже созданную группу, галочки, соответствующие выбранным для данной группы правам, расставятся автоматически.

Вы можете изменить набор прав, поставив дополнительные галочки. Следует иметь в виду, что права пользователя не могут быть меньше, чем права группы, в которую он входит.

Подробнее о настройке прав

Вы можете самостоятельно расширять и редактировать набор полей, необходимых для заполнения при создании/редактировании пользователя. Для этих целей существует модуль Шаблоны данных.
