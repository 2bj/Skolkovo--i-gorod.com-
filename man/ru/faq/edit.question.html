На этой странице вы можете изменить 3 основных поля:

Название — можете изменить имя для пары вопрос-ответ.

Вопрос — в этом поле можно отредактировать вопрос, на который требуется дать овтет.

Ответ — впишите или измените сюда ответ.

Вопросы заданные посетителями с сайта по умолчанию скрываются. На этой странице можно добавить ответ и сделать вопрос видимым для всех посетителей сайта, включив опцию «Активность».
