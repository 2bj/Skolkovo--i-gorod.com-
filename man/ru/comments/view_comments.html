Здесь вы можете видеть таблицу со списком комментариев в хронологической последовательности, начиная от самого последнего. Для добавления поля в таблицу нужно кликнуть на пиктограмму и выбрать нужное поле. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов.

Вы можете скрыть или удалить любой комментарий при помощи всплывающих пиктограмм. Вы также можете отредактировать любой комментарий по ссылке с него или при помощи пиктограммы .

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
