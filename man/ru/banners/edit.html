Основные параметры

  • Название баннера — можно изменить название баннера.
  • Выберите тип данных — можно сменить тип: картинка, Flash или HTML-код.
  • Активен — включает и выключает активность баннера.
  • Теги — баннер будет показан только на тех страницах, которые помечены соответствующими тегами.
  • Описание — пояснительный текст, предназначенный для администраторов. Выводится в списке баннеров в качестве примечания.

Параметры перехода

  • URL — адрес страницы, которая откроется при нажатии на баннер.
  • Открыть в новом окне — страница, на которую ведет баннер, откроется в новом окне.

Параметры показа

  • Количество показов — показывает текущую статистику по показам данного баннера на сайте.
  • Максимальное количество показов — введите сюда число показов, после которого будет автоматически отключен баннер.
  • Количество переходов — оказывает число переходов с баннера на страницу рекламодателя на текущий момент.
  • Дата начала и окончания показа — позволяет автоматически включить и отключить баннер в заданную дату. Датой начала показа автоматически ставится дата создания баннера, это число можно изменить. Если не задана дата окончания показа, то баннер будет показываться без ограничения по срокам.
  • Показывать пользователям с тегами — данный баннер будет показан пользователям, "собравшим" соответствующие теги.
  • Подробнее о работе тегов читайте в топике Как пользоваться тегами.

Разделы отображения

  • Страницы — те разделы сайта, в которых баннер будет появляться. Нажмите иконку и выберите нужные разделы из дерева сайта.
  • Место показа — выберите места на странице, где должен появляться баннер.

Time-таргетинг

Time-таргетинг предназначен для того, чтобы показывать баннер на сайте только в определенное время. Модуль позволяет показывать баннеры: по числам месяца, по месяцам, по дням недели, по времени суток.
