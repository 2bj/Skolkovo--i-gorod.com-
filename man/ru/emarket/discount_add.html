Введите название скидки и выберите тип скидки.

После выбора типа скидки, в нижней секции нужно будет указать предварительные параметры скидки и указать правила валидации скидки.

Скидки на заказ

Модификатор цены — для скидок на заказ можно выбрать фиксированный размер скидки, или процент от суммы.

Правила валидации

Исходные условия: cкидка будет действовать на все заказы, для любого пользователя, все время.

Доступные ограничения (показать)

  • На сумму заказа — скидка будет установлена только на заказы больше определенной суммы.
  • На группу пользователей — скидка будет установлена только на заказы, сделанные пользователями определенной группы.
  • На пользователей — скидка будет установлена только на заказы, сделанные пользователями из заданного списка.
  • На сумму покупок пользователя — скидка будет установлена только на заказы пользователей, чья общая сумма заказов больше указанной суммы, с учетом всей истории заказов пользователя.
  • На временной промежуток — скидка будет установлена на временной интервал.

Скидки на товары каталога

Модификатор цены — для скидок на товары можно выбрать только процент от суммы.

Правила валидации

Исходные условия: cкидка будет действовать на все товары, для любого пользователя, все время.

Доступные ограничения (показать)

  • На определенные товары — скидка будет установлена только на указанные товары.
  • На группу пользователей — скидка будет установлена на товары, для пользователей из определенной группы.
  • На пользователей — скидка будет установлена на товары, для пользователей из указанного списка.
  • На связанные товары — скидка будет действовать только, если указанные товары уже лежат в корзине.
  • На сумму покупок пользователя — скидка будет установлена на товары только для пользователей, чья общая сумма заказов больше указанной суммы, с учетом всей истории заказов пользователя.
  • На временной промежуток — скидка будет установлена на временной интервал.

Нажмите «Добавить», чтобы перейти к настройкам выбранных условий скидки.
