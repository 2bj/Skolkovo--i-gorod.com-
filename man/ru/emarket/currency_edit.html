Название - можно отредактировать название для валюты.

Код валюты - можно изменить буквенный код валюты. По этому коду происходит автоматическое обновление курса валюты.

Номинал - можно изменить номинал валюты. Это поле определяет единицы расчета в этой валюте для магазина.

Курс - можно отредактировать курс валюты для магазина.

Префикс - можно изменить префикс валюты. Префикс обычно ставится перед суммой (например: $15).

Суффикс - можно изменить суффикс валюты. Суффикс обычно ставится после суммы (например: 15 руб).
