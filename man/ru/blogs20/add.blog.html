Введите Название нового блога. Это название будет отображено на сайте и видно всем посетителям

Список друзей. В этом списке вы можете выбрать тех пользователей, которые будут являться вашими друзьями. В постах можно будет выставлять статус доступа «для друзей» - такие посты будут видны только друзьям. Внимание - если добавите Гостя в друзья, то пост «только для друзей» будут видеть все.

Остальные поля являются стандартными для страниц UMI.CMS.

После заполнения необходимых полей нажмите на кнопку Добавить в любом из блоков.
