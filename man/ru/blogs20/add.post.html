Введите Название новой публикации. Это название будет отображено на сайте и видно всем посетителям

В поле Контент введите текст новой публикации.

Видимо только друзьям — такие публикации будут видны только друзьям. Внимание — если добавите Гостя в друзья, то пост «только для друзей» будут видеть все.

Остальные поля являются стандартными для страниц UMI.CMS.

После заполнения необходимых полей нажмите на кнопку Добавить в любом из блоков.
