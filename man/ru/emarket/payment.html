В этой вкладке перечислены способы оплаты, доступные на вашем сайте. Эти способы будут предложены покупателю на последнем шаге оформления заказа.

Для добавления нового способа оплаты нужно воспользоваться ссылкой «Добавить способ оплаты».

Способы оплаты можно отсортировать по типу оплаты, экспортировать в файл CSV или импортировать из файла CSV.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
