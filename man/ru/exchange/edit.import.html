Название — можно отредактировать название для схемы импорта.

Формат данных — можно изменить формат данных, который используется при импорте.

Файл данных — можно выбрать или закачать новый файл для импорта. Чтобы файлы были доступны в выпадающем списке, необходимо загружать их в папку ~/files/filesToImport/

Раздел сайта — можно изменить раздел сайта, в который импортируются данные.
