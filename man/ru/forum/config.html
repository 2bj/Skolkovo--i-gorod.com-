Количество объектов на странице — этот параметр ограничивает количество конференций, топиков и сообщений, выводимых на одну страницу. Если сообщений больше, список разбивается на несколько страниц.

Не публиковать без модерирования — если стоит эта отметка, то все новые сообщения попадают в категорию «Скрытых», а на сайте появляются только после проверки модератором.

Разрешить добавлять сообщения незарегистрированным пользователям — если отметка стоит, то незарегистрированные посетители сайта могут оставлять сообщения. Если снята, то сообщения смогут оставлять только зарегистрированные пользователи, входящие в эту группу.

Сортировать темы по последнему сообщению — при включенной опции темы в списке на сайте будут сортироваться по времени написания последнего сообщения в теме.

Добавлять новые темы в рассылку — при помощи этой опции у вас есть возможность включать в рассылку уведомления о создании новых тем в форуме. Для этого выберите из списка название рассылки, в которую будут попадать сообщения о новых темах.
