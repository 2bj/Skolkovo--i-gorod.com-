Название — укажите название для новой схемы импорта.

Формат данных — выберите формат данных, который будет использоваться при импорте.

Файл данных — выберите или закачайте файл для импорта. Чтобы файлы были доступны в выпадающем списке, необходимо загружать их в папку ~/files/filesToImport/

Раздел сайта — выберите раздел сайта, в который будут импортироваться данные.
