Для осуществления операции экспорта выберите тип данных для экспорта:

  • Прайс-лист 1С (CommerceML версии 1.0) - цены для товаров для импорта в «1С: Предприятие 7.7» «Торговля и склад» 8.7.
  • Прайс-лист 1С (CommerceML версии 2.0) - цены для товаров для импорта в «1С: Предприятие 8» редакция «Управление торговлей» 10.3 и выше.
  • Каталог 1С (CommerceML версии 2.0) - каталог товаров для импорта в «1С:Предприятие 8» редакция «Управление торговлей» 10.3 и выше.
  • Яндекс.Маркет (YandexML) - данные для импорта каталога товаров в Яндекс.Маркет.
  • Данные в формате UMIDump 1.0 - данные во внутреннем формате UMI.CMS.

После чего выберите разделы каталога, предназначенные для экспорта и нажмите на кнопку "экспортировать".

Откроется страница с результатом экспорта. Для импорта в Яндекс.Маркет необходима ссылка этой страницы, в остальных случаях надо скопировать содержимое страницы. В браузере Opera необходимо копировать исходный код страницы.
