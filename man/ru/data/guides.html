Справочник — это тип данных, который объединяет объекты этого типа в один раздел. Введены для создания выпадающих списков.

Содержание справочника — это объекты справочника (элементы выпадающего списка).

Обычно справочники используются для того, чтобы задать несколько значений, либо для того, чтобы дать пользователю возможность выбирать из нескольких существующих вариантов.

Пример: хотим, чтобы у товара было свойство "Поддерживаемые форматы аудио".

Решение: создаем справочник "Форматы аудио", заносим в него: "mp3", "wav", ... . Создаем у типа товара поле, которое использует в качестве справочник "Форматы аудио", и выбираем нужные значения.

Для просмотра и редактирования элементов справочника вызовите контекстное меню нажатием правой кнопкой мыши (в Опере Alt+левый клик) и выберите пункт "Содежимое справочника".

Для этих же целей можно воспользоваться всплывающей пиктограммой .

В большинстве случаев используется одно поле и значение, однако, можно вводить дополнительные атрибуты, например, для вывода пиктограмм.
