Свойства типа

Название типа — здесь задается имя типа данных

Назначение типа — эта настройка устанавливает соответствие этого типа с базовым типом. Устанавливается автоматически, если создается подтип. Не меняйте значение этого поля у существующих типов, если не не знаете, что делаете.

Общедоступный — это действует только в том случае, если отмечена галочка "Использовать как справочник". Это означает, что этот тип можно будет редактировать в списке справочников, и что можно динамически добавлять туда значения (под выпадающим списком появится поле для ввода текста).

Можно использовать как справочник — позволяет использовать объекты этого типа данных в качестве справочника. То есть при создании полей типа «выпадающий список» или «список с множественным выбором», мы сможем выбрать этот тип данных в качестве справочника и при редактировании страницы этого типа, в поле «выпадающий список» можно будет выбрать один или несколько объектов справочника.

Подробнее о справочниках читайте в топике Выпадающий список.

Поля и группы полей

Если до этого момента вы уже работали с системой UMI.CMS и создавали страницы сайта в визуальном редакторе модуля Структура, то названия полей покажутся вам сразу знакомыми: в поле TITLE вы вписывали название страницы, заполняли meta-поля и так далее. Теперь вы видите, что тот набор полей, с которым вы работали в редакторе, задается именно здесь.

Каждая группа полей содержит ссылку «Добавить поле», при помощи которой можно вставить новое поле в существующую группу. Новое поле появится внизу списка.

Редактирование и добавление полей

Название поля — название поля, отображаемого при добавлении/редактировании объекта сайта.

Идентификатор — при помощи него мы можем вывести значение поля на сайт. Работает идентификатор точно так же как и макрос: чтобы значение поля появилось на сайте, необходимо записать его в виде %identifikator% и вставить либо в шаблон, либо на страницу сайта. Идентификаторы можно использовать в качестве параметров для других макросов.

Подсказка — информация, введенная в это поле, будет показана в административной части рядом с самим полем при наведении на название поля.

Тип поля — позволяет выбрать необходимый тип поля из выпадающего списка.

Использовать справочник — справочник используется только в тех видах полей, где присутствует список. Таких типов два: выпадающий список и список с множественным выбором. Читайте о создании и назначении справочника в соответствующих топиках документации.

Видимое — при включенном флажке поле будет отображаться в свойствах объекта. При необходимости поле можно отключить, сняв флажок.

Индексировать — при включенном флажке значение поля будет индексироваться поисковой системой по сайту.

Использовать в фильтрах — при включенном флажке посетитель сайта будет иметь возможность искать объект, используя фильтрацию по этому полю.

Обязательное — Поставьте эту галочку, если необходимо, чтобы это поле стало обязателььным к заполнению. В случае, если поле не будет заполнено, при сохранении страницы будет выведена ошибка о необходимости заполнить соответствующее поле.

Ссылка вверху «Добавить группу полей» позволит создать еще один «контейнер», который затем можно наполнить новыми полями. Новая группа полей появится внизу списка.

Редактирование и добавление групп полей

Название группы — название поля, отображаемого при добавлении/редактировании группы объектов сайта.

Идентификатор — при помощи него мы можем вывести содержимое этой группы на клиентскую часть сайта.

Видимое — при включенном флажке группа полей будет отображаться в свойствах объекта. Соответственно при желании группу можно временно отключить, сняв флажок.

Вы также можете перемещать поля и группы между блоками, перетаскивая их с помощью функции drag&drop.
