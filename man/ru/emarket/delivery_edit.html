Название — здесь можно изменить название для способа доставки.

Тип доставки — выбранный при создании тип доставки. В зависимости от выбранного типа, ниже может быть доступен различный набор параметров доставки для расчета ее стоимости.

Возможные параметры доставки:

Самовывоз — никаких настроек не требуется.

Курьерская доставка — укажите стоимость доставки курьером в соответствующем поле. Если это необходимо, укажите минимальную стоимость для бесплатной доставки в поле «Минимальная стоимость заказа».

Почта России — выберите вид отправления и способ пересылки. Если это необходимо, установите опцию «С объявленной ценностью».
