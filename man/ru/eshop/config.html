Настройка магазина

Ответственный менеджер — администратор, отвечающий за работу с заказами. Ему будут приходить уведомления по почте о каждом заказе, сделанном с сайта.

Имя, от которого отправлять письмо — это имя будет присутствовать в поле "От кого" во всех письмах модуля интернет-магазина.

E-mail, от кого отправлять письмо — этот адрес будет виден адресатам, получающим письма интернет-магазина.

Тема письма о новом заказе (отправляется заказчику) — этот заголовок будет присутствовать в теме письма, отправленном покупателям магазина после сделанного заказа.

Тема письма о новом заказе (отправляется менеджеру) — этот заголовок будет присутствовать в теме письма, отправленном ответственному менеджеру магазина после сделанного заказа.

Тема письма о изменении статуса заказа (отправляется менеджеру) — этот заголовок будет присутствовать в теме письма, отправленном менеджеру в случае, если статус заказа был изменен.

Разрешить оплату без проверки заказа — при включённой опции покупатель будет иметь возможность оплатить товар без подтверждения наличия в магазине, в момент отправки заказа.

Максимальное количество сравниваемых товаров — задает ограничение на максимальное количество товаров, которое может добавить посетитель в блок "сравнение товаров".

Настройки для платежной системы ChronoPay

Введите в этой секции данные полученные при регистрации на сайте системы Chronopay.

В случае если выключена опция «Не требуется подтверждение предварительных результатов оплаты», будет произведена проверка: в течение определенного времени система ChronoPay будет проверять факт оплаты, а затем передаст системе UMI.CMS результат. Если эта опция выключена, то статус заказа поменяется на «Оплачен успешно» сразу после оплаты.
