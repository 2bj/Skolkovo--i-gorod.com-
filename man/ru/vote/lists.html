Модуль Опросы предназначен для проведения на сайте опросов и голосований.

На этой странице можно видеть список созданных опросов, хранящихся в системе

Здесь можно добавить новый опрос или редактировать уже существующие. Чтобы создать новый опрос, нажмите на ссылку Добавить опрос над списком. Откроется окно редактирования нового опроса.

Нажатие на ссылку с названием опроса откроет страницу редактирования его свойств.

Существующие опросы можно блокировать (скрывать с сайта).

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
