Спасибо Вам за выбор UMI.CMS!

UMI.CMS предъявляет определенные требования к аппаратному и программному обеспечению.
    * Apache с установленным mod_rewrite или nginx + php-fastcgi. При использовании nginx для кэширования данных мы рекомендуем устанавливать модуль mod_realip для Apache версии 1.х или mod_rpaf для Apache версии 2.x.
    * PHP 5.1.6 и выше, кроме PHP 5.2.0 и 5.3.0.
    * на PHP не должен быть установлен патч Suhosin Patch
    * Библиотеки zLib, GD, libxml, mb_string, iconv, xsl
    * MySQL версии не ниже 4.1 с InnoDB
    * UMI.CMS устанавливается в корневую директорию (обычно htdocs, www или public_html), которая должна быть доступна на чтение и запись. UMI.CMS не может быть установлена в подпапку (www.domain.ru/directory/), поскольку в таком случае невозможна работа mod_rewrite.
    * В настройках Apache должна быть включена директива AllowOverride со значением "All" или "Options FileInfo AuthConfig", и при этом директива AccessFileName имеет значение .htaccess
    * В настройках PHP объем памяти, выделяемой на выполнение скриптов (параметр memory_limit), должен составлять не меньше 16Мб
Обратите внимание - UMI.CMS необходимо устанавливать в пустую корневую папку сайта.

Список рекомендуемых хостингов вы можете посмотреть на нашем сайте http://www.umi-cms.ru/support/umi_cms_php5_hosting/
Для нормальной работы UMI.CMS достаточно аппаратных ресурсов большинства хостеров. При тестировании UMI.CMS выдержала более миллиона запросов в сутки на простом однопроцессорном сервере P4-1600 1G RAM, при этом работая параллельно с десятками других сайтов.
Мы не рекомендуем выбирать младшие тарифные планы виртуального хостинга, поскольку на них действуют значительные ограничения на доступ к ресурсам сервера. Перед выбором хотинг-провайдера и тарифного плана уточните, какие ограничения действуют данном тарифном плане - возможно, эти ограничения будут тормозить работу сайта. 

Установка:
*	Необходимо загрузить распакованные файлы по FTP в режиме Binary на сервер в корневой каталог вашего сайта.
*	Запустите установщик. Для этого в адресной строке браузера наберите адрес будущего сайта и "/install.php" в конце (например, "www.site.ru/install.php").
*	Далее следуйте пошаговым инструкциям установщика.

Шаг 1. Лицензионное соглашение
Вы должны ознакомится и подтвердить Ваше согласие с Лицензионным соглашением.

Шаг 2. Проверка настроек сервера
Установщик сам проверит настройки сервера, требуемые для работы UMI.CMS.
Если какие-то настройки не соответствуют требуемым, установщик сообщит об этом. В этом случае свяжитесь с хостинг-провайдером и попросите его изменить настройки сервера в соответствии с требованиями UMI.CMS.

Шаг 3. Проверка лицензионного ключа
Введите лицензионный ключ, полученный от разработчиков или дилеров системы. После проверки ключа (необходимо соединение с интернетом) процесс установки продолжится.

Если лицензионный ключ введен неверно, установщик оповестит Вас об этом. Проверьте корректность введенных данных, в случае проблем обратитесь к продавцу системы. 


Шаг 4. Выбор пакета установки
На этом этапе выберите пакет (дистрибутив) установки. Обычно в папке «packages» находится только один пакет, тогда просто нажмите кнопку «Продолжить».

Шаг 5. Ввод параметров базы данных
Введите параметры соединения с БД MySQL. Эти данные можно узнать у своего хостинг-провайдера при покупке хостинга или настроить самомтоятельно, если ваш хостинг-провайдер позволяет это сделать.

Шаг 6. Распаковка архива и создание базы данных
На этом этапе установщик сам распакует архив, корректно разместит нужные файлы и создаст базу данных.
Если в процессе работы возникнет ошибка (например, из-за неправильно введенных параметров соединения с базой данных), установщик сообщит об этом.

Шаг 7. Создание пароля супервайзера
После окончания установки будет предложено создать пароль супервайзера. Запомните этот пароль! Супервайзер — это самый главный администратор сайта, имеющий неограниченный доступ ко всем настройкам, функциям и содержимому.

Шаг 8. Выбор демосайта.
На этом шаге можно выбрать один из четырех вариантов установки: либо один из трех вариантов демосайта, либо "пустую" систему.

Документацию по UMI.CMS вы можете найти здесь: http://umi-cms.ru/support/docs/
При возникновении вопросов обращайтесь в службу заботы о клиентах по адресу: http://umi-cms.ru/support/


Желаем продуктивной работы!
