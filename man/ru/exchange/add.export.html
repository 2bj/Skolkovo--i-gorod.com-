Название — укажите название для новой схемы экспорта.

Формат данных — выберите формат данных, который будет использоваться при экспорте.

Разделы сайта — выберите разделы сайта, из которых будут экспортироваться данные.

Время кеширования — здесь можно указать время, в течении которого последние результаты для экспорта будут храниться в кеше и при запросе извлекаться оттуда. Это позволяет снизить нагрузку на систему в случае больших объемов экспортируемых данных.
