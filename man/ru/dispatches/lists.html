Здесь вы можете видеть таблицу со списком рассылок, созданных на сайте. Для добавления поля нужно кликнуть на пиктограмму в правом углу. Набор добавляемых полей задается базовыми типами модуля. Весь список может быть отсортирован по любому из столбцов.

Вы можете посмотреть список сообщений, входящих в эту рассылку, зайдя в диалог редактирования рассылки — либо нажав на название рассылки, либо на всплывающую пиктограмму .

Над списком находится ссылка для добавления новой рассылки. При нажатии на нее откроется окно с настройками новой рассылки.

Начиная с версии 2.8, можно редактировать свойства элементов непосредственно из таблицы со списком: для этого наведите курсор на поле, значение которого необходимо изменить, и нажмите на всплывающую пиктограмму . Для сохранения изменений нажмите Enter, для отмены изменений — Esc.
