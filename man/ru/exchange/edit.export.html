Название — можно изменить название для схемы экспорта.

Формат данных — можно изменить формат данных, который используется при экспорте.

Разделы сайта — можно изменить разделы сайта, из которых экспортируются данные.

Время кеширования — здесь можно указать или изменить время, в течении которого последние результаты для экспорта будут храниться в кеше и при запросе извлекаться оттуда. Это позволяет снизить нагрузку на систему в случае больших объемов экспортируемых данных.
