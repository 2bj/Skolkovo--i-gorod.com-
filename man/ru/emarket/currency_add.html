Название - укажите название для новой валюты.

Код валюты - укажите буквенный код валюты. По этому коду будет происходить в дальнейшем автоматическое обновление курса валюты.

Номинал - укажите номинал валюты. Это поле определяет единицы расчета в этой валюте для магазина.

Курс - укажите курс валюты.

Префикс - укажите префикс валюты. Префикс обычно ставится перед суммой (например: $15).

Суффикс - укажите суффикс валюты. Суффикс обычно ставится после суммы (например: 15 руб).
