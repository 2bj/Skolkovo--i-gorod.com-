Здесь вы можете видеть список элементов выбранного справочника.

Можно отредактрировать любое значение при помощи всплывающей пиктограммы непосредственно на этой странинце, либо перейти на страницу редактирования значения, по ссылке с названия элемента.

Для добавления нового элемента необходимо воспользоваться ссылкой «Добавить наименование справочника». Добавление произойдет без перезагрузки страницы: будет добавлен элемент "(Без названия)", дать название которому можно будет, либо непосредственно с этой страницы (), либо со страницы редактирования элемента.

Список элементов также может быть экспортирован и импортирован в формате CSV.
