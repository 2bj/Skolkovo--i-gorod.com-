Здесь вы можете посмотреть или изменить свойства и содержимое данной рассылки.

Название будет показано посетителям на сайте, на странице выбора рассылок для подписки. Служебное описание рассылки предназначено для пометок администратора.

Ссылка «Добавить сообщение» открывает страницу редактирования свойств нового сообщения. После заполнения свойств новое сообщение будет добавлено в очередь на отправку.

По ссылке «Архив выпусков» можно просмотреть список всех отправленных ранее сообщений.

«Подписчики» на рассылку — пройдя по этой ссылке вы увидите список всех подписчиков данной рассылки.

Для заполнения рассылки сообщениями нажмите на кнопку «Заполнить новостями» в секции Сообщения нового выпуска. Все новые новости из ленты, заданной в поле Связано с лентой новостей, будут добавлены списком в сообщения нового выпуска.

Вы можете отредактировать отдельные сообщения из списка или удалить, при помощи соответствующих ссылок.

Если список сообщений готов, нажмите «Отправить» для рассылки писем подписчикам.
