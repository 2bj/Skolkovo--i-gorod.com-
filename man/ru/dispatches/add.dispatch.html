Введите в соответствующих полях название и описание новой рассылки. Название будет показано посетителям на сайте, на странице выбора рассылок для подписки. Служебное описание рассылки предназначено для пометок администратора.

Выберите в поле Связано с лентой новостей привязку к новостной ленте, — эта привязка будет использоваться для заполнения сообщениями данной рассылки.

Нажмите «Добавить», для того, чтобы приступить к заполнению рассылки сообщениями.
